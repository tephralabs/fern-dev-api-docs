# Customer Verification (KYC/KYB)

## Overview

Customer verification is a critical compliance process that ensures secure and lawful financial transactions. Fern's Customer Verification API provides comprehensive Know Your Customer (KYC) and Know Your Business (KYB) verification capabilities for both individual and business customers.

This documentation provides detailed technical guidance for implementing customer verification workflows using Fern's API. Whether you're verifying individual customers or complex business entities with beneficial owners, these guides will help you build compliant verification systems.
